Team Project (6명) 2025.04 ~ 2025.06

다구독 (Subsub Clipclop)

OTT 구독 고객 관리 B2B SaaS 플랫폼

OTT 서비스를 운영하는 기업이 고객 행동 데이터를 업로드하고, 구독 현황 분석, 고객 세그먼트 추출, 이탈 예측 및 맞춤 전략 추천까지 자동화된 분석과 시각화를 제공받을 수 있는 통합 플랫폼입니다.

주요 기능

  • 로그인 및 권한 관리 (JWT 인증)
  • 고객 데이터 업로드 (Excel/CSV)
  • 구독 현황 및 세그먼트 분석
  • 코호트 분석 및 이탈 예측 (SHAP)

담당 역할

  • Spring Security 및 JWT 기반 보안 아키텍처 설계
  • Redis를 활용한 Refresh Token 관리 시스템 구현
  • Flask 기반 세그먼트 분류 모델 개발
  • Spring Boot와 Flask 연동, S3 저장/호출 기능 개발
  • AWS EC2에 Flask 환경 구축 및 배포

기술 스택

Spring Boot 3 Spring Security React Flask MariaDB Redis AWS EC2/S3
Backend Frontend ML/AI

Team Project (5명) 2025.02 ~ 2025.03

허겁직업 (HugUpJigUp)

취업 준비생을 위한 멘토링 및 정보 공유 커뮤니티

취준생들에게 최신 정보와 노하우를 공유하고 전문가 멘토링 및 네트워킹을 지원하는 커뮤니티 플랫폼입니다. 모든 유저가 멘토와 멘티로 상호작용할 수 있는 다분야 상호 멘토링 시스템이 특징입니다.

주요 기능

  • 이메일 로그인/회원가입 (OTP 인증)
  • 멘토/멘티 모집 게시판 (비밀 댓글)
  • 멘토링 후기 및 평점 시스템

담당 역할

  • 유저 프로필 CRUD API 개발
  • Redis 기반 Refresh Token 도입 전략 제안
  • 프로필/멘토링/게시판 UI 구현

기술 스택

Spring Boot Spring Security MariaDB Redis

Listify 🎵

음악 추천 및 플레이리스트 관리 웹 서비스

Spotify API와 연동하여 음악을 추천하고, 사용자가 직접 플레이리스트를 만들고 관리할 수 있는 풀스택 웹 프로젝트입니다.

주요 기능

  • 회원가입 및 로그인 (JWT 인증)
  • 음악 추천 및 검색 (Spotify API)
  • 플레이리스트 생성/관리
  • 음악 미리듣기

담당 역할

  • Flask 기반 Spotify API 연동 백엔드 API 설계
  • MySQL DB 모델링 및 CRUD 로직 구현

기술 스택

Flask React TypeScript MySQL Docker Spotify API
