Team Project 2024

다구독 (Subsub Clipclop)

OTT 구독 고객 관리 B2B SaaS 플랫폼

OTT 서비스를 운영하는 기업이 고객 행동 데이터를 업로드하고, 구독 현황 분석, 고객 세그먼트 추출, 이탈 예측 및 맞춤 전략 추천까지 자동화된 분석과 시각화를 제공받을 수 있는 통합 플랫폼입니다.

SHAP 기반의 머신러닝 예측 모델을 활용해 이탈 위험 고객을 사전 식별하고 맞춤형 마케팅 전략을 제공합니다.

주요 기능

  • 로그인 및 권한 관리 (JWT 인증)
  • 고객 데이터 업로드 (Excel/CSV)
  • 구독 현황 및 세그먼트 분석
  • 코호트 분석 및 이탈 예측
  • 결제 시스템 연동

기술 스택

Spring Boot 3 JPA Spring Security React TypeScript TailwindCSS Flask SHAP MariaDB Redis AWS EC2 GitHub Actions

02
Team Project 2024

허겁직업 (HugUpJigUp)

취업 준비생을 위한 멘토링 및 정보 공유 커뮤니티

취업 준비 과정에서 막막함을 느끼는 취준생들에게 실질적인 도움을 제공하기 위해, 최신 정보와 노하우를 공유하고 전문가 멘토링 및 네트워킹을 지원하는 커뮤니티 플랫폼입니다.

기존 취준 커뮤니티와 멘토링 시스템을 결합하여 모든 유저가 멘토와 멘티로 상호작용할 수 있는 다분야 상호 멘토링 시스템이 특징입니다.

주요 기능

  • 이메일 로그인/회원가입 (OTP 인증)
  • 회원 프로필 및 멘토링 히스토리 관리
  • 멘토/멘티 모집 게시판 (비밀 댓글)
  • 자유 게시판 및 취업정보 게시판
  • 멘토링 후기 및 평점 시스템

기술 스택

Spring Boot Spring JPA Spring Security MariaDB Redis Swagger UI

03
Team Project 2024

Listify 🎵

음악 추천 및 플레이리스트 관리 웹 서비스

Spotify API와 연동하여 음악을 추천하고, 사용자가 직접 플레이리스트를 만들고 관리할 수 있는 풀스택 웹 프로젝트입니다.

React 기반의 프론트엔드와 Flask 기반의 백엔드, MySQL 데이터베이스, Docker를 활용한 배포 환경을 갖추고 있습니다.

주요 기능

  • 회원가입 및 로그인 (JWT 인증)
  • 음악 추천 및 검색 (Spotify API)
  • 플레이리스트 생성/관리
  • 음악 미리듣기 (Spotify 웹 플레이어)
  • 사용자 프로필 관리

기술 스택

Flask Python React TypeScript Vite MySQL Docker Spotify API
